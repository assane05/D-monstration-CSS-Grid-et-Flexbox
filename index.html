CSS Grid

Les grilles CSS permettent de créer des mises en page en deux dimensions (lignes et colonnes).

Flexbox

Flexbox est idéal pour les mises en page à une dimension (ligne OU colonne).

Modifiez les propriétés du conteneur flex:

1
2
3
4

Code CSS actuel:

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: flex-start;
  align-items: stretch;
  flex-wrap: wrap;
}
CSS Grid
  • Pour les mises en page en 2 dimensions (lignes ET colonnes)
  • Idéal pour les structures complexes à plusieurs niveaux
  • Parfait pour les mises en page globales de page
  • Permet le positionnement précis des éléments dans une grille
  • Assure des espacements et alignements cohérents
Flexbox
  • Pour les mises en page en 1 dimension (ligne OU colonne)
  • Idéal pour les composants d'interface utilisateur
  • Parfait pour les barres de navigation, listes, etc.
  • Gère bien la distribution d'espace
  • S'adapte facilement aux différentes tailles d'écran
